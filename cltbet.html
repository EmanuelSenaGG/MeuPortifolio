Casa de Apostas - CLTBET

Sistema desktop desenvolvido em Java com Swing como projeto final para a disciplina de Programação Orientada a Objetos.

Sobre o Projeto

O desafio deste projeto acadêmico era construir uma aplicação interativa com Java Swing, consumindo dados de uma fonte externa. Minha solução envolveu a integração com a API Football para buscar informações em tempo real. Além disso, para facilitar a colaboração em equipe, criei uma imagem Docker do banco de dados. Essa abordagem simplificou o setup do ambiente e garantiu consistência. O projeto foi um sucesso, culminando em uma aplicação funcional e na nota máxima.

Repositório do projeto: CLTBET no GitHub

Principais Tecnologias

  • Java
  • JDBC
  • Swing
  • Docker
  • PostgreSQL
  • API Football

Demonstração e Funcionalidades

Além de permitir o gerenciamento das apostas pelo usuário Admin, o sistema também possibilita que o usuário comum selecione múltiplos jogos, com a odd sendo atualizada automaticamente, realize diversas apostas e acompanhe seu histórico. O sistema conta ainda com mecanismos de depósito e saque, além de creditar ou debitar o saldo do usuário conforme o resultado das apostas.

A seguir, destaco os principais aprendizados:

  • Orientação a Objetos
  • Prática com containers Docker
  • Interfaces desktop com Swing
