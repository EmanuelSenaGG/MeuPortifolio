Sistema de Ouvidoria

Sistema Web completo para gestão de ouvidoria em uma instituição pública, cobrindo todo o ciclo de vida das manifestações.

Sobre o Projeto

Durante minha atuação em um projeto para o setor público, liderei o desenvolvimento completo de um novo sistema para a área de ouvidoria. Fui responsável por todo o ciclo do projeto, desde a análise das regras de negócio e modelagem do banco de dados, até a implementação full-stack (back-end e front-end). A solução foi entregue com sucesso e recebeu feedback positivo tanto da equipe interna quanto do setor de ouvidoria, que validou a alta qualidade e aderência do sistema às suas necessidades.

Principais Tecnologias

  • ASPNET Web Forms
  • ADO.NET
  • SQL Server

Funcionalidades e Aprendizados

Um sistema de ouvidoria completo com tudo que se tem direito. Diversas formas de contato, manifestações que podem ser anônimas ou não, envio de documentos, além de acompanhamento das manifestações enviadas pelo CPF.

A seguir, destaco os principais aprendizados:

  • Arquitetura de Software
  • Engenharia de Software
  • Modelagem de Banco de Dados
