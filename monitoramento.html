Sobre o Projeto

Na empresa em que atuo, fui designado para desenvolver um sistema de monitoramento de APIs. O sistema tem como objetivo monitorar o status de APIs de maneira automatica e realizar a atualização em tela, além de realizar testes de endpoints e fornecer relatórios detalhados sobre o desempenho das APIs, como cabecalhos, tempo de resposta, tamanho da resposta e status da requisição. O sistema é desenvolvido em ASPNET Web Forms e realiza a verificação do status de cada endpoint a cada 1 minuto, os endpoints pertecem a varios sistemas, um deles sendo o gateway e os outros sendo microserviços, todos sendo desenvolvidos com ASPNET Core Web API. O JavaScript foi utilizado para realizar os testes individuais, com os parametros personalizados pelo usuario, por meio do fetch, além de obter os dados da requisição, montar o relatorio e mostrar em tela, como se fosse um Swagger por exemplo.

Funcionalidades

Atualização automática do status de cada API a cada 1 minuto, com a possibilidade de realizar testes individuais de cada endpoint, com os parâmetros personalizados pelo usuário.
A seguir, destaco os principais aprendizados:

  • Programação Assincrona
  • Manipulação JSON em JavaScript
  • Requisição Fetch
  • Configuração de política de CORS
  • Manipulação de estado de objeto
