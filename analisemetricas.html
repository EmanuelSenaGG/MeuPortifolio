Sobre o Projeto

A aplicação consiste em captar e exportar as métricas do gateway utilizando o OpenTelemetry, que é um pacote de código aberto para observabilidade. As métricas são exportadas para o Prometheus, um sistema de monitoramento e alerta também de código aberto. O Grafana é utilizado para criar dashboards em tempo real, permitindo a visualização das métricas captadas pelo Prometheus. Todas as métricas são capturadas e exportadas por meio das configurações realizadas na classe Program da aplicação. O resultado final foi muito satisfatório e elogiado pelo meu superior, tendo em vista que, antes desse sistema, obtínhamos apenas informações sobre o uso de memória e CPU de maneira não muito confiável. O sistema que desenvolvi permitiu uma maior compreensão sobre o que estava acontecendo com o gateway e possibilitou a tomada de medidas e a adoção de estratégias de performance baseadas em dados muito mais consistentes.

Funcionalidades

Monitoramento nos dashboard das mais diversas métricas possiveis como por exemplo, quantidade de requisições por endpoint, quantidade de erros, memoria consumida da aplicação, memoria gasta durante uma requisição, memoria do GC Heap, memoria total limpa pelo Garbage Colector, uso da CPU, entre varias outras.
